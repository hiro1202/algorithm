# algorithm

Kotlinでアルゴリズムを学ぶためのリポジトリ。

## 実装アルゴリズム

| カテゴリ | アルゴリズム |
|---|---|
| ソート | バブルソート、カクテルソート |
| 二分探索 | 配列の二分探索、一般化二分探索（最小の true を探索） |
| 再帰 | 合計計算、フィボナッチ数列、ユークリッドの互除法 |
| 全探索 | 線形探索、最小値探索 |
| 動的計画法 | 蛙跳び問題 |
| 貪欲法 | コイン問題 |

## 実行

```bash
./gradlew run        # サンプル実行
./gradlew test       # テスト実行
./gradlew build      # ビルド
```

## 技術スタック

- Kotlin / Gradle
- JUnit 5 / AssertJ / JaCoCo（カバレッジ 80% 以上）
