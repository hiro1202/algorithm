# algorithm

アルゴリズムを学ぶためのリポジトリ。複数言語で実装しています。

## 実装アルゴリズム

| カテゴリ | アルゴリズム |
|---|---|
| ソート | バブルソート、カクテルソート、挿入ソート、マージソート、ヒープソート、クイックソート、バケットソート |
| 二分探索 | 配列の二分探索、一般化二分探索（最小の true を探索） |
| 再帰 | 合計計算、フィボナッチ数列、ユークリッドの互除法 |
| 全探索 | 線形探索、最小値探索 |
| 動的計画法 | 蛙跳び問題、ナップサック問題 |
| 貪欲法 | コイン問題 |
